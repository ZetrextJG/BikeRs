# BikeRs
Repo for PDU project nr. 3

## Software requirements

- Python > 3.8
- R / 3.6


## How to start

### Install necessary python packages

```bash
pip install -r requirements.txt
```

### Install necessary R packages

```bash
Rscript install_packages.R
```

### Download CitiBike data

For Ubuntu:
```bash
python3 ./get_data.py
```

For Windows:
```bash
python get_data.py
```
